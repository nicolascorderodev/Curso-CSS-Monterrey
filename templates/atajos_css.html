Atajos de clases e IDs

VS Code nos facilita agregar clases e IDs a elementos nuevos. Para esto podemos escribir el nombre del tipo de elemento seguido de un "." o un "#" y el nombre de la clase o ID.

Ejemplo para clases

h1.clase.doble-clase

Ejemplo para IDs

p#nombre-de-id

Podemos agregar múltiples clases al elemento iniciando cada nombre por un ".". A diferencia de las clases, no podemos agregar más de un ID a un elemento.

Ejemplo usando varias clases y otros atajos para agregar atributos y contenido dentro del mismo atajo.

header.principal>h1.titulo[style="color: blue;" width="200px"]{Soy un h1}+p.parrafo+ul#nicolas>li
