En CSS para acceder a elementos id o classse usa (#) para ID y (.) para class.

Si utilizamos un ID(atributo global) con un nombre especifico no se va a poder utilizar ese mismo ID con otros elementos. Para el atributo class se pueden utilizar varias clases dentro de esa misma etiqueta separadas por espacio para que lo logre identificar como clases distintas.
