Estilos en cascada en CSS(Cascading Style Sheets)

Algo más que se se debe tener en cuenta es que en CSS los cambios se aplican en orden de cascada(de arriba a abajo). Esto quiere decir que si se modifica una propiedad de una división al principio y se vuelve a modificar al final del CSS, el estilo asignado será el último.

Ejemplo

Este texto es azul

Esto es un texto con dos clases.

Este ejemplo nos muestra esto de dos maneras:

  1. La clase morado se modifica primero con el color blanco, pero al final con el color morado y esto es lo que se visualiza.
  2. La clase rojo también se aplica al segundo texto en el código HTML, pero como la clase morado es la última del archivo CSS, esta es la que se ve.

Divs en Visual Studio Code

VS Code nos permite escribir elementos <div> de manera rápida, para esto solo hay que escribir un punto . seguido del nombre de la clase que queremos asignarle a nuestra división, de esta manera:

Ejemplo de atajo usando clases

.division1.division2


Para un ID, entonces en lugar de un punto utilizamos un #.

Ejemplo de atajo usando ID´S

#division1
