Elementos dentro de una división en CSS(Cascading Style Sheets)

Podemos ser aún mas especificos y seleccionar elementos dentro de otros elementos. Esto se hace escribiendo selectores separados por espacios.

Ejemplo con elementos dentro de una división

Este encabezado no es azul

Este texto es azul

Este texto tampoco es azul.

En este ejemplo, el único texto azul es el del párrafo que está dentro de un elemento con la clase azul. El encabezado no es azul, pues, aunque está dentro de la división, no está dentro de un párrafo. El último párrafo tampoco es azul porque no está dentro de un elemento con la clase azul.
