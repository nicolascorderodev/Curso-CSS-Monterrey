Introducción a CSS (Cascading Style Sheets)

Con HTML puro podemos hacer una página web bastante sencilla pero no se ve del todo bien, o al menos no se ve como una página web moderna. Agregando CSS podemos cambiar colores, cambiar el orden de los elementos, tipo de letra, color de letra entre muchas otras cosas que podemos hacer agregando CSS.

Las competencias que voy a adquirir en este módulo son las siguientes:

  • Conocer lo básico sobre el lenguaje de estilo CSS.
  • Identificar tamaños y responsividad adecuados.
  • Integrar diferentes fuentes de texto, colores y rellenos.
  • Modificar el posicionamiento de elementos
